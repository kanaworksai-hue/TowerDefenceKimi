# 僵尸塔防游戏

一款基于HTML5 Canvas和JavaScript开发的策略塔防游戏。建造防御塔，抵御源源不断的僵尸入侵！

## 游戏简介

《僵尸塔防》是一款经典的策略塔防游戏。玩家需要在僵尸入侵的路径上建造和升级防御塔，阻止僵尸到达终点。游戏采用波次进攻模式，每波僵尸会越来越强大，考验你的资源管理和策略布局能力。

### 游戏特色

- **4种防御塔**：基础塔、狙击塔、速射塔、溅射塔，各有特色
- **4种僵尸类型**：普通僵尸、快速僵尸、坦克僵尸、飞行僵尸
- **升级系统**：每座塔可升至3级，解锁特殊能力
- **波次系统**：20波渐进式难度，挑战你的极限
- **末日科技风**：独特的视觉风格和特效

## 本地运行

### 方式一：直接打开

1. 克隆或下载本仓库
2. 直接用浏览器打开 `index.html` 文件

### 方式二：使用本地服务器

```bash
# 使用 Python 3
python -m http.server 8000

# 或使用 Node.js (需安装 http-server)
npx http-server -p 8000

# 或使用 VS Code Live Server 插件
```

然后访问 `http://localhost:8000`

## 操作说明

### 基本操作

| 操作 | 说明 |
|-----|------|
| 鼠标左键 | 选择/建造/升级防御塔 |
| 鼠标右键 | 取消选择 |
| 空格键 | 显示所有塔的攻击范围 |
| P键 | 暂停/继续游戏 |

### 游戏界面

- **顶部状态栏**：显示生命值、金币、波次、得分
- **底部塔面板**：点击选择要建造的防御塔类型
- **游戏控制**：开始波次、暂停、加速按钮

### 防御塔说明

| 防御塔 | 成本 | 特点 |
|-------|------|------|
| 基础塔 | 100金币 | 均衡型，攻速快 |
| 狙击塔 | 250金币 | 高伤害，远射程 |
| 速射塔 | 200金币 | 极快攻速，适合清理小怪 |
| 溅射塔 | 350金币 | 范围伤害，适合群攻 |

### 游戏流程

1. **准备阶段**：观察地图，规划防御布局
2. **建造阶段**：点击塔按钮，在可建造区域放置防御塔
3. **波次开始**：点击"开始波次"按钮，僵尸开始进攻
4. **战斗阶段**：防御塔自动攻击范围内的僵尸
5. **升级优化**：点击已建造的塔进行升级或出售
